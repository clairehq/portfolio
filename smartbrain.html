Smartbrain Face Detection

  • Date: May—June, 2020

  • My Role: Web developer

  • Type: Individual Project

  • Tools: VS Code, Git, SQL(TablePlus)

Project Description

This project is a face detection website. The front-end is built with react and uses a free image recognition API. The website has its server, using node.js and express.js. It also has its own database that users can register, sign in, and sign out.

Project Process

build the website

First, I decided what components I need to built and then built the components using React framework and used React Router for the page routing. In order to make the face detection function work, I configured a free image recognization API through Clarifai Face Detection Model.

connect the front-end with the back-end

After I built the website interface, I used node.js and express.js to set up a server that can communicate with the front-end. Finally, I created a database using TablePlus to memorize users' email and passwords.

Open the Website

Reflection
